# Welcome to Markdown Presentations

Turn your markdown files into beautiful presentations.

---

## Getting Started

It's simple to create presentations:

1. Create a markdown file in the `presentations` folder
2. Separate slides with `---` (horizontal rule)
3. Use standard markdown syntax
4. Click on any presentation to start

---

## Features

- **Keyboard Navigation**: Use arrow keys to navigate
- **Smooth Animations**: Beautiful slide transitions
- **Markdown Support**: Full markdown rendering
- **Code Highlighting**: Perfect for technical presentations
